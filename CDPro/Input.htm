How to Analyze a given Protein CD data Using CDPro?

  • Copy all files into the working directory. This can be done by Clicking on the name of the file and saving the file on your computer. Once saved these files can be copied to a diskette. The programs are written in FORTRAN and in a PC they work in MSDOS environment.
  • Running a given program: To run a program goto to the directory from the file manager and double-click on the executable. This will pop-up a DOS-Window and the instructions in the window are self-explanatory. Alternatively, one can open a DOS-window, change directory to where the programs reside and type the program name.

    •  
  • Create the input file INPUT. One can use the template of the sample input file (INPUT.SMP) and edit (using a text editor like WORDPAD) the required fields. The CD data in De uints is required and should be entered (one can use cut-paste operation) in the data fields in decreasing wavelength order (e.g., 260 to 190 nm).

This program accepts information about CD data file interactively in a MSDOS window.  Input file for CRDATA (provided in CRDATA.INP) looks like:
