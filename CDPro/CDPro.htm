Different Options (Reference Sets) Available in CDPro:
The reference set of proteins from different sources are combined to create a large reference set of 56 CD spectra. Depending on the wavelength range of interest the number of proteins in the reference set could be as large as 56. These are used in any of the three programs by changing the IBasis value (formarly Basis_1) in the INPUT file (varies from 1 to 10).

Both SOLUBLE and MEMBRANE Proteins are now included in the reference set. Options 9 and 10 include 13 MEMBRANE proteins along with  37  and 43 soluble protiens, respectively, from options 3 and 4.

Results from IBasis values 1, 3, 4, 6 and 7 should be comparable as the secondary structure fractions they estimate are identical (Kabsch & Sander, 1983, DSSP assignments as modified by Sreerama et al). IBasis = 2 (King & Johnson, 1999) and IBasis = 5 (DSSP, as modified by Sreerama & Woody, 1994) use a different assignment of secondary structure.
IBasis = 8 uses tertiary class specific reference set that has to be created by executing program: CLUSTER.

Program and Data Files in CDPro:
Download Instructions:

These executables were created for an IBM-compatible PC. The SELCON3 and CONTINLL source codes can be compiled and used on any computer, but CDSSTR cannot be since it relies upon GetTim, Seed and Random routines of MSFortran. The CDSSTR source code for a unix machine is available. Other two programs run on unix machine as is (Just change the suffix from FOR to f; e.g., SELCON3.FOR to selcon3.f).

The files can be downloaded either individually or as a zip-file. The zip-file is: CDPro.zip(Double-Click on the name and save it on your PC. Copy it to the required directory and unzip it.)

It was created by WINZIP utility. If you donot have WINZIP utility you can download it for free from INTERNET. Other softwares are also available for UNZIPPING the CDPro.zip file. If you have Windows XP, the XP operating system dows it for you. You can also use PKZIP25.EXE. Command for unzipping the files is: PKZIP -extract CDPro.zip which implodes 42 files. The html files (index.htm, Guide.htm, Input.htm, Output.htm and CDPro.htm) have information about CDPro software.

One can also download each file separately if there is a problem, as instructed below.

  1. Create a directory ( C:/CDPRO if you want to use the RunCDPro.html from Netscape Navigator) in your computer hard drive (or insert a diskette in your floppy drive)
  2. Double-Click on the file names that are in columns Data Files, Source/Executable Code and Sample I/O files. This will either open a dialog box (for executables and large files) or bring the file in to the browser (select save option from file menu).
  3. Save the file on your hard drive (or diskette).
