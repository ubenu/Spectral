OUTPUT Summary from CDPro:

  • Once the INPUT file is ready run the programs SELCON3, CDSSTR and CONTINLL in any order. The output files are created and the final result from each program is written in PROTSS.OUT  --- Individual program outputs (SELCON3.OUT, CDSSTR.OUT and CONTINLL.OUT) can also be examined in case of problems. (The OUTPUTs now also contain the RMSD between the calculated and experimental spectra.)
  • Program CLUSTER modifies the  INPUT file for reading tertiary class specific reference protein set and creates the required files. The option IBasis is 8 for this reference set. (OUTPUT: CLUSTER.OUT)
  • A New parameter NRMSD is printed. This gives the Normalized RMS deviation between the calculated and experimental spectra. (Also called: Goodness of the FIT.)
  • Care must be taken while using the RMSD and NRMSD values from different reference sets and different programs. Generally, these values (both RMSD and NRMSD) are the smallest for CONTINLL and the largest for SELCON3 (CDSSTR has intermediate value) for a given spectrum and a given reference set.

OUTPUT from Each Program: (SELCON3, CDSSTR, and CONTINLL and CLUSTER)
    Output from each program contains details of how the final solution is arrived at. Most of the output is self-explanatory. If the secondary structures estimated include the estimation of distorted helix and strand fractions then the number of segments of helix and strand are also calculated.

    Some of the features and formatting in the output from the current version of CDPro may be different from what is below (Please see SAMPLE outputs for the three programs: SELCON3.OUT  CONTINLL.OUT  CDSSTR.OUT.) There may be some additional information, but the nature and quality of the values are the same.
